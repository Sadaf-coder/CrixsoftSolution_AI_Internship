# AI-Powered Customer Support Chatbot

A rule-based AI chatbot designed using Natural Language Processing (NLP) concepts to handle customer inquiries, provide support, and engage users in human-like conversations.

## Features
- Contextual intent-matching using keywords.
- Randomized human-like responses for static queries.
- Handles common support cases: Refunds, Timings, and Human Agent transfers.
- Clean and interactive Command Line Interface (CLI).
